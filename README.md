# configFiles

.bashrc, .emacs, .tmux.conf, etc.

## Setup

When cloning this repo to a new server, run `setup.sh` to install
packages (ESS, Julia Emacs mode, etc) and to set up symlinks from my
home directory to the local copy of this repo.
